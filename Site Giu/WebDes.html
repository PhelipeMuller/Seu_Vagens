Web Design:

Molde de Portfólio

   

Este mesmo portfólio que você está visitando agora, foi desenvolvido por mim e outros 3 colegas, porém em forma de molde, podendo ser reutilizado facilmente para outra pessoa, sem que o usuário perca a oportunidade de ter um portfólio com a própria cara.

Nós, para este projeto, aprendemos HTML, CSS e Javascript e vários conceitos de Co-Design.
