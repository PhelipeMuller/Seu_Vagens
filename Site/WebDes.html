Molde de Portfólio

   

Este mesmo portfólio que você esta visitando agora, foi desenvolvido por mim e outros 3 colegas, porem em forma de molde, podendo ser reutilizado facilmente para outra pessoa, sem que o usuario perca a oportunidade de ter um portfólio com a propria cara.

Nós, para este projeto, aprendemos HTML, CSS e Javascript e varios conceitos de Co-Design.
