Inspermon

   

A aula de Design de Software tinha como objetivo estabelecer o contato dos alunos com a programação em python partindo do mais básico possível uma vez que era uma matéria completamente nova. .

O Inspermon foi o segundo projeto realizado na matéria e além de um resultado muito satisfatório também possuía um nível de complexidade interessante para quem não havia tido muito contato com programação.

O projeto consistia em realizar uma série de objetivos quem estavam no plano do projeto, da maneira que o aluno quisesse fazer, assim possuindo alguns limitadores mas já abrindo mais as opções para realizar o projeto, tendo inclusive um objetivo que era de surpreender o professor.

A programação do projeto foi algo que demandou tempo, pois muitas coisas podiam ser colocadas no projeto e nem todas eram convenientes, adicionando assim um trabalho em entender a melhor maneira de realizar o projeto e tentar surpreender o professor.

Projeto final de design de software

O projeto final de Design de Software tinha como objetivo utilizar todo o conhecimento aprendido em programação até o momento e também inovar. Sendo o último projeto o grau de complexidade era maior, e também era maior a liberdade para escolher o que realizariam uma vez que o tema era livre.

Antes de iniciar o projeto todos os grupos tiveram uma pequena reunião com o professor para decidir o tema do programa a ser feito, com variações entre programas uteis para o dia a dia, sites e até mesmo jogos.

Pelo grau de complexidade do que estava sendo feito foram disponibilizadas aulas estúdio para realizar o projeto, com o professor constantemente checando o progresso de cada grupo para que ninguém acabasse ficando perdido

No final foi feita uma feira aonde os alunos apresentaram seus projetos e o deixaram em exposição para quem quisesse testar, uma parte muito relevante para o projeto por conta dos feedbacks, principalmente para grupos que tinham o desejo de continuar com o projeto
